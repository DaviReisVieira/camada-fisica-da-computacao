## Instruções

Para executar o Projeto 4 e 5, abra 2 Promts de Comando e execute em um:

```cmd
python Server.py
```

Após inicalizar o Server, execute em outro Prompt:

```cmd
python Client.py
```

Observações: As portas do Client e do Server devem ser diferentes.
